# 🇧🇷 Brazilian E-Commerce Sales Dashboard

## 📌 Overview

This project analyzes the Brazilian E-Commerce (Olist) dataset using SQL and Power BI to uncover sales trends, customer behavior, payment patterns, and product performance. The goal is to build an interactive business intelligence dashboard that helps stakeholders monitor key business metrics and make data-driven decisions.

---

## 📂 Dataset

- **Source:** Olist Brazilian E-Commerce Dataset (Kaggle)
- **Records:** 100K+ Orders
- **Period:** 2016–2018

---

## 🛠️ Tools & Technologies

- SQL (MySQL)
- Power BI
- DAX
- Python (Data Cleaning)

---

## ❓ Business Questions

- What is the total revenue?
- How many orders and customers are there?
- Which states generate the highest revenue?
- Which product categories perform the best?
- What are the most common payment methods?
- How does revenue change over time?
- What is the average customer review score?
- What is the average delivery time?


---

## 📈 Key KPIs

- Total Revenue
- Total Orders
- Total Customers
- Average Order Value
- Average Review Score

---

## 💡 Key Insights

- São Paulo generated the highest revenue.
- Bed Bath Table was the best-selling product category.
- Credit Card was the most frequently used payment method.
- Average customer review score was approximately **4.1**.
- Average delivery time was approximately **13 days**.
